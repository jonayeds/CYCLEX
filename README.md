# By-Cycle Store
## React - Redux Frontend for a By-cycle Store
--> <a href="https://bi-cycle-store-shad-cn.vercel.app/">Live Website Link</a>


## Project Features
- Creating a By-Cycle
- Get all By-Cyles
- Get a single By-Cycle
- Update a By-Cycle
- Delete a By-Cyle
- Order a By-Cycle
- Get total revenues
- Integrated Stripe Payment Method
- Cancel order
- BLock Customer
